🎯 SUR VOTRE PC (Windows) - Pousser sur GitHub
═══════════════════════════════════════════════════════════

1. Committer :
   git commit -m "Fix yt-dlp + Ajout logs interactifs et debug complet"

2. Pousser sur GitHub :
   git push origin main


🚀 SUR LE VPS - Appliquer les Corrections
═══════════════════════════════════════════════════════════

1. Se connecter :
   ssh user@votre-vps-ip

2. Aller dans le dossier :
   cd ~/botmusic

3. Mettre à jour depuis Git :
   git pull origin main

4. Rendre les scripts exécutables :
   chmod +x fix-ytdlp.sh logs-vps.sh quick-commands.sh update-vps.sh

5. Corriger yt-dlp :
   ./fix-ytdlp.sh

6. Redémarrer le bot :
   pm2 restart botmusic

7. Vérifier les logs :
   pm2 logs botmusic


📊 ACCÉDER AUX LOGS À TOUT MOMENT
═══════════════════════════════════════════════════════════

Menu interactif complet :
   ./logs-vps.sh

Commandes rapides :
   ./quick-commands.sh logs        (Logs en temps réel)
   ./quick-commands.sh errors      (Erreurs uniquement)
   ./quick-commands.sh status      (Statut)
   ./quick-commands.sh check       (État système)

PM2 direct :
   pm2 logs botmusic               (Temps réel)
   pm2 logs botmusic --err         (Erreurs)
   pm2 monit                       (Monitoring)


🎵 TESTER LE BOT
═══════════════════════════════════════════════════════════

Sur Discord :
   /play nom de la chanson

Sur le VPS (surveiller) :
   pm2 logs botmusic
